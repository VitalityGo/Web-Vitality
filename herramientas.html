Herramientas Utilizadas

Descubre las herramientas que hicieron posible el desarrollo de Vitality Go.

Firebase

Firebase es una plataforma de desarrollo de aplicaciones móviles y web creada por Google. Ofrece bases de datos en tiempo real, autenticación de usuarios, hosting web y más.

Visual Studio Code

Un editor de código ligero y rápido desarrollado por Microsoft, utilizado por programadores para escribir y depurar código en diversas plataformas.

Adobe Illustrator

Un software de diseño gráfico utilizado para la creación de ilustraciones, logotipos y gráficos vectoriales.

IDX Workbench

Una herramienta de desarrollo avanzada que facilita la integración de flujos de trabajo en la nube con soporte para múltiples plataformas.

Adobe Photoshop

Un software de edición de imágenes líder en la industria, utilizado para la manipulación y mejora de fotografías digitales.
